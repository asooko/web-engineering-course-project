# Web Engineering Course Project

## Overview

This project was developed as part of a Web Engineering course at Hochschule Niederrhein.

The website was designed using HTML, CSS, and JavaScript and demonstrates fundamental front-end web development concepts.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Responsive web design
- Interactive user interface
- Multi-section webpage layout
- Client-side functionality using JavaScript

## Learning Outcomes

Through this project I gained practical experience in:

- Front-end web development
- Website structure and styling
- JavaScript programming
- User interface design
